0. 基本流程和工具¶

李航给出的机器学习定义是：计算机基于数据构建概率统计模型并运用模型对数据进行预测和分析的学科。

这里的模型是通用的，但是参数不是，对于不同数据有不同的参数。在很多方法，比如梯度下降和K-Means，都是通过不断的优化来得到最优参数，给定可能的一些值，给定评价标准，让机器选择最合适的参数，称之为学习。

机器学习的学习值得到底是：给定空白模型，不断优化得到最优模型的能力，还是得到最优模型后，通过不断的新数据，将过去的最优模型优化为更优模型的能力？我认为都说得通。

1.4 一些有趣的探索：邻居¶

In [19]:
k_range = range(1,20)
scores = []

for k in k_range:
    knn = KNeighborsClassifier(n_neighbors = k)
    knn.fit(X_train, y_train)
    scores.append(knn.score(X_test, y_test))

plt.figure()
plt.xlabel('k')
plt.ylabel('accuracy')
plt.scatter(k_range, scores)
plt.xticks([0,5,10,15,20]);

KNN原理是，将空间中任意点看作一个cell，然后将共同被标记为同一类的点看作是此类的范围。K的意思是，在进行类的划分的时候(比如说，对于一个新的值)，使用K个临近点（欧几里得距离到这个点最小的k个邻居们）的类别，通过多数表决的方式进行预测。

对于较小的K值，具有较大的估计误差以及较小的近似误差，只有当新的点很接近这些老点的时候，才能精准预测。换句话说，过拟合。

对于较大的K值，具有较大的近似误差和较小的估计误差，这是典型的拟合不足。对于接近总个数的K的选择，会使用所有的点来进行决策，这导致了必定会被分配到占据优势地位的类中去。
