            本章概览

            这些是Django的基础知识，它并算不上一个教程或者指南，但是，麻雀虽小，五脏俱全。我在写完这篇笔记后就写出了一个用作链接跳转的Web服务，配合CGI跑在服务器上，已经工作几个月了。本文从配置环境、设置项目、设计模型、ADMIN可视化管理、模板语法，文件和使用、URL跳转管理、视图这几个方面介绍了Django。本文基于2.0.2版本的Django。
